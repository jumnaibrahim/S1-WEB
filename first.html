Types of CSS

  1. Inline CSS

    2. Inline CSS is applied directly to
an HTML element using the style attribute.

  2. Internal CSS

    3. Internal CSS is defined within the style 
tag in the HTML document's head section.

  3. External CSS

    4. External CSS is defined in a separate
file with a .css extension, linked to the 
HTML document using the link tag.
